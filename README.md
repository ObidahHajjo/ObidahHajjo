## 💻 Featured Projects

### 🚗 Carpooling App (Covoiturage)

A full-stack web application that allows users to **search, publish, and manage trips**, with real-time chat between drivers and passengers.

**Key features:**
- 🔍 Trip search (city + date filters)
- 📅 Booking & reservation system
- 💬 Real-time chat (Laravel Reverb + WebSockets)
- 🔐 JWT authentication with refresh flow
- ⚙️ Admin & role-based access control

👉 Frontend: React + TypeScript + Tailwind  
👉 Backend: Laravel + PostgreSQL + Redis  
👉 Infra: Docker + Traefik + Cloudflare  

[🔗 View Back-end(Api) Repository](https://github.com/ObidahHajjo/covoit-api)  
[🔗 View Front-end Repository](https://github.com/ObidahHajjo/covoit-front)
